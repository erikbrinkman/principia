space1

    Space things apart on a line

    Parameters

    • elements: [number, number][]

      An array of line segments that are spaced apart so they no long overlap. Each element is an array of [position, width], where position is the smallest point on the line. If you have a number line like 0--1--2..3..4--5, the elements array would like [[0, 2], [4, 1]].
